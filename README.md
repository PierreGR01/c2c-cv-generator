# C2C Tenders App

Application interne Camptocamp pour générer des CV PDF à la charte Camptocamp, ciblés sur un appel d'offre, et tenir à jour les fiches maîtres collaborateurs — authentification OAuth par utilisateur, données sur Google Drive.

## Lancer l'app

- **Windows** — double-clique sur `lancer.bat`
- **Mac** — double-clique sur `lancer.sh` (1re fois : clic droit → Ouvrir)

Le code applicatif, la configuration et la documentation détaillée vivent dans [`codebase/`](codebase/) :

- [`codebase/SETUP.md`](codebase/SETUP.md) — installation et utilisation au quotidien
- [`codebase/SETUP-OAUTH.md`](codebase/SETUP-OAUTH.md) — configuration Google Cloud (admin)
- [`codebase/README.md`](codebase/README.md) — stack technique
